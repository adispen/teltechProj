# Install Instructions

#### Requires: python, python-devel, pip, and virtualenv

* Clone locally
* ``` $ cd teltechProj ```
* ``` $ virtualenv venv ```
* ``` $ source venv/bin/activate ```
* ``` $ pip install -r requirements.txt ```
* ``` $ python app.py ```
* navigate to ```localhost:5000``` or ```localhost:5000/chat```
